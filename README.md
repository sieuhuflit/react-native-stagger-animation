# react-native-stagger-animation

React native stagger animation

## Installation

```sh
npm install react-native-stagger-animation
```

## Usage

```js
import StaggerAnimation from "react-native-stagger-animation";

// ...

const result = await StaggerAnimation.multiply(3, 7);
```
